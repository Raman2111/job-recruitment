### Prerequisites for running the Project.

> You need to node.js version >= 14

> An active internet connection

### Steps to Run project.

#### 1. copy .env file to => config/
### To run backend
#### 2. Run `npm install` [for first Time only]
#### 3. Run `npm run server`
### To run frontend
#### 1. Run `cd client`
#### 2. Run `npm install` [for first Time only]
#### 3. Run `npm start`npm
